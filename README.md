
# 🏠 House Price Prediction

## 📌 Project Overview

This project aims to predict house sale prices using Machine Learning techniques. The dataset was cleaned, analyzed, and prepared for model building through data preprocessing and exploratory data analysis (EDA).

## 🎯 Objective

To build a regression model that can accurately predict house prices based on various property features such as area, quality, location, garage capacity, basement area, and other housing characteristics.

## 📂 Dataset

* Dataset: House Prices Dataset (Ames Housing)
* Total Features: 77
* Target Variable: `SalePrice`

## 🧹 Data Cleaning

The following preprocessing steps were performed:

* Handled missing values
* Checked and removed duplicate records
* Validated data types
* Analyzed categorical and numerical features
* Created a cleaned dataset for further analysis

## 📊 Exploratory Data Analysis (EDA)

Performed exploratory data analysis to understand the dataset and identify important features affecting house prices.

### Key Analysis Performed

* SalePrice distribution analysis
* Correlation analysis
* Feature relationship with SalePrice
* Categorical feature analysis
* Outlier detection
* Neighborhood and quality-based price analysis

### Key Insights

* OverallQual shows a strong positive relationship with SalePrice.
* GrLivArea significantly impacts house prices.
* Garage-related features contribute to higher sale prices.
* Neighborhood plays an important role in determining property value.
